# نظام إدارة المزادات - Admin Panel

نظام إدارة المزادات مبني بـ React + TypeScript مع واجهة إدارة حديثة.

## 🚀 المميزات

### التقنيات الأساسية

- **React 18** - أحدث إصدار من React
- **TypeScript** - تطوير آمن بالأنواع
- **Vite** - أداة بناء سريعة
- **Tailwind CSS** - إطار عمل CSS حديث

### واجهة المستخدم

- **Radix UI** - مكونات واجهة مستخدم
- **shadcn/ui** - مكتبة مكونات جميلة
- **Lucide React** - أيقونات جميلة
- **Framer Motion** - رسوم متحركة سلسة

### إدارة البيانات

- **React Query** - إدارة حالة الخادم
- **Axios** - عميل HTTP
- **React Hook Form** - إدارة النماذج
- **Zod** - التحقق من البيانات

## 📁 هيكل المشروع

```
src/
├── components/         # مكونات واجهة المستخدم
├── pages/             # صفحات التطبيق
│   ├── Login.tsx      # صفحة تسجيل الدخول
│   └── Dashboard.tsx  # لوحة التحكم
├── contexts/          # سياق React (المصادقة، الثيم)
├── schemas/           # مخططات التحقق
├── types/             # تعريفات TypeScript
├── services/          # خدمات API
└── hooks/             # Hooks مخصصة
```

## 🛠️ التشغيل

### المتطلبات

- Node.js 18+
- npm أو yarn

### التثبيت والتشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل الخادم التطويري
npm run dev

# بناء للإنتاج
npm run build
```

## 🔑 تسجيل الدخول

### بيانات الدخول الافتراضية:

- **رقم الهاتف:** 01001234567
- **كلمة المرور:** admin123456

### API Endpoint:

```
POST http://localhost:4000/api/v1/admin/login
Content-Type: application/json

{
  "phoneNumber": "01001234567",
  "password": "admin123456"
}
```

### Response:

```json
{
  "access_token": "JWT_TOKEN_HERE",
  "admin": {
    "_id": "687a79078f2fa5b78920da63",
    "name": "مدير النظام",
    "email": "admin@mazad.com",
    "phone": "01001234567",
    "role": "admin",
    "active": true
  }
}
```

## 📱 الصفحات

### 1. صفحة تسجيل الدخول (`/login`)

- نموذج تسجيل دخول باللغة العربية
- التحقق من البيانات
- دعم إظهار/إخفاء كلمة المرور
- تصميم مطابق للمخطط المطلوب

### 2. لوحة التحكم (`/dashboard`)

- نظرة عامة على النظام
- إحصائيات المزادات والمستخدمين
- معلومات المدير
- تسجيل الخروج

## � المصادقة والحماية
